## What this is

`warrenvilleproud.com`: a free, community-run local hub for Warrenville, Illinois. Events calendar + twice-weekly blog. The goal is to genuinely help the town (awareness, growth) while serving as a soft, credible marketing surface for Matt's print shop.

## The model (why it can pay for itself)

Useful free content brings in local readers. Local readers are valuable to local businesses. **Sound & Fury Print Shop** (Matt's shop, soundandfuryprint.com) is the **founding sponsor**, present on every page. The **Local Business Spotlight** rotates, so other Warrenville businesses can be sold sponsorships over time. Trust comes first; the sponsorship works because the site is actually helpful.

## How it is built

Static site generated by `scripts/build_site.py` from three data files:

- `data/events.json`: upcoming events (refreshed weekly from public sources).
- `data/posts.json`: blog posts (full content lives here).
- `data/sponsors.json`: founding sponsor + rotating spotlight (`spotlight_index`).

Run `python3 scripts/build_site.py` to regenerate all HTML. Add `--rotate` to advance the spotlight. The generator also writes `sitemap.xml` and `robots.txt`. Set `REF_DATE=YYYY-MM-DD` to control which events count as upcoming (used for previews).

Do not hand-edit the generated `.html` files; edit the data and rebuild.

## Voice and rules

- Warm, local, neighborly. Useful first, promotional second.
- Sound & Fury sponsorship is always present (ribbon, footer, per-post note). Never ship without it.
- Every event links back to its official source. Never copy descriptions word for word.
- No em dashes (workspace rule). The build strips them as a backstop.
