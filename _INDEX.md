# warrenville-proud

Source for **warrenvilleproud.com**: a free, community-run local hub for Warrenville, Illinois (events calendar + twice-weekly blog), with **Sound & Fury Print Shop** as founding sponsor and a rotating Local Business Spotlight that can be sold to other businesses. Self-sustaining model: useful free content draws local readers, local sponsors fund it. Created **2026-06-17**.

Static site generated by a Python script from JSON data. Refreshed automatically by two weekly scheduled tasks. Has its own `CLAUDE.md`.

## Files

| File | What it is |
|------|------------|
| `index.html`, `events.html`, `blog/` | Generated site pages. Do not hand-edit; edit the data files and rebuild. |
| `sitemap.xml`, `robots.txt` | Generated SEO files. |
| `assets/style.css` | Shared stylesheet (prairie-green civic theme, light mode, responsive). |
| `data/events.json` | Upcoming Warrenville events. Refreshed weekly from public sources. Each event links to its official source. |
| `data/posts.json` | Blog posts (full HTML body lives here). Newest first. |
| `data/sponsors.json` | Founding sponsor (Sound & Fury) + rotating spotlight list and `spotlight_index`. |
| `scripts/build_site.py` | Static site generator. `python3 scripts/build_site.py` rebuilds everything; add `--rotate` to advance the spotlight. `REF_DATE=YYYY-MM-DD` controls which events count as upcoming. No dependencies. |
| `WEEKLY_TASK.md` | The automation runbook the two scheduled tasks follow (Monday: calendar refresh + Week Ahead post; Thursday: feature post + spotlight rotation). Lists the public event sources. |
| `PUBLISH.md` | Plain-English one-time deploy guide (GitHub Pages + GoDaddy DNS + redirecting the other 5 domains). |
| `CLAUDE.md` | Auto-loaded site memory: model, build process, voice, and rules. |
| `pipeline/post-history.md` | Anti-repeat ledger of published post topics. |
| `.gitignore` | Standard ignores. Folder is a git repo. |
